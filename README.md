##  Git Workflow Summary
To complete this assignment, I followed a structured Git workflow to track my progress and version history:

### 1. Repository Setup
- Created a new repository on GitHub and cloned it locally.
- Initialized the project structure with `README.md`, `assistant_draft.md`, and `final_version.md`.
- **Commands**:
  - `git add .`
  - `git commit -m "repository setup"`

### 2. Assistant-Created Draft
- Used **GitHub Copilot Chat** to generate the initial content for `assistant_draft.md`.
- Documented the exact prompt and AI performance notes at the bottom of the file.
- **Commands**:
  - `git add assistant_draft.md`
  - `git commit -m "assistant-created draft"`

### 3. Manual Revision
- Copied the draft to `final_version.md` and performed manual edits to add personal details and improve clarity without using AI.
- **Commands**:
  - `git add final_version.md`
  - `git commit -m "your own edited version"`

### 4. Final Synchronization
- Updated the `README.md` with project details and the walkthrough video link.
- **Commands**:
  - `git add README.md`
  - `git commit -m "final README update with video link"`
  - `git push origin main`
